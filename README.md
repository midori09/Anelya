# Book Tracker (Трекер прочитанных книг)

**Автор:** [Твое Имя и Фамилия]

## Описание программы
Приложение с графическим интерфейсом (GUI) для ведения списка прочитанных книг. 
Позволяет сохранять название, автора, жанр и количество страниц. 
Данные хранятся локально в формате JSON.

## Функционал
- Добавление новых книг с валидацией данных.
- Просмотр списка всех книг в удобной таблице.
- Фильтрация по жанру.
- Быстрая фильтрация книг объемом более 200 страниц.
- Автоматическое сохранение и загрузка данных из файла `books.json`.
