## Описание программы
**Currency Converter** — это десктопное приложение с графическим интерфейсом (GUI), предназначенное для мгновенной конвертации валют. 
Программа использует актуальные данные о курсах валют через внешний API, поддерживает валидацию ввода и сохраняет историю операций в локальный файл.

### Основные функции:
*   **Конвертация в реальном времени:** Получение свежих курсов через ExchangeRate-API.
*   **История операций:** Автоматическое сохранение всех расчетов в файл `history.json` и их отображение в интерфейсе.
*   **Валидация данных:** Проверка на ввод положительных чисел.
*   **Современный интерфейс:** Использование библиотеки `customtkinter` для комфортной работы.

---

## Как получить API-ключ
Для работы приложения необходим ключ от сервиса [exchangerate-api.com](https://exchangerate-api.com):
1. Зарегистрируйтесь на сайте (бесплатного тарифа достаточно).
2. Подтвердите email.
3. Скопируйте ваш персональный **API Key** из панели управления.
4. Вставьте полученный ключ в код программы (переменная `self.api_key`).

## Примеры использования (Тесты)
*   **Тест 1 (Успех):** Выбрать USD -> RUB, ввести "100", нажать кнопку. Результат отобразится на экране, запись появится в таблице истории.
*   **Тест 2 (Ошибка ввода):** Ввести "-50" или "текст". Приложение выдаст предупреждение о необходимости ввода положительного числа.
*   **Тест 3 (Автономность):** Закрыть и снова открыть приложение. История предыдущих конвертаций должна загрузиться из `history.json` автоматически.
